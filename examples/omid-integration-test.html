Tester Checklist — What to verify
  1. OMID feature advertised: The creative's log panel should show feature:omid → Supported ✓ and the OMID Active badge should appear in the creative.
    This confirms OmidCompatBridge.getFeatureName() is wired into Container:init supportedFeatures.
  2. Script injection: Open DevTools → the creative iframe's document (srcdoc origin). Its <head> should contain injected <script> tags for the OM SDK service script, session client, and SHARC SDK files before any creative-authored tags.
    This confirms OmidCompatBridge.injectIntoMarkup() ran and the container used srcdoc instead of src.
  3. OMID session started: After clicking Load Ad, the browser console inside the creative frame should log [SHARC OMID Bridge] Session started.
    If the OM SDK scripts are not available (404), the bridge logs a warning and continues without crashing — the ad still renders.
  4. Play / Pause / Quartiles: In the creative, click Play. Verify that omid:play, omid:firstQuartile, omid:midpoint, omid:thirdQuartile, and omid:complete appear in the creative's event log in order. Pause and resume mid-playback — verify omid:pause and omid:resume appear.
  5. Auto-pause on page hide: Click Simulate Page Hide while the ad is playing. The creative should auto-pause and log omid:auto-pause. Click Simulate Page Show — it should auto-resume and log omid:auto-resume.
    This confirms the stateChange → hidden path.
  6. Skip event: Click Skip in the creative. Verify omid:skip appears in the creative log. This maps to mediaEvents.skipped() in the OM SDK.
  7. Clean close: Click Close Ad (publisher side) or the ✕ button in the creative. Verify:
    • The event log shows Container:close → resolve.
    • The ad slot returns to the empty placeholder.
    • The browser console logs [SHARC OMID Bridge] Session finished.
  8. Extension destroy: After close, OmidCompatBridge.destroy() is called if defined. Currently the bridge has no persistent resources, but the hook is wired — verify no JS errors appear in the console after close.
